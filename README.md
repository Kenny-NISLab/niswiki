# niswiki

## Command

```zsh
# Check Version
git --version
npm -v

# Git Command
git clone https://github.com/kogepanh/niswiki/
git pull origin 
git checkout -b feature/#ticket
git push origin feature/#ticket

# Install Package
npm install

# Run Local Server
npm start
```

## 補足

`/docs/.nojekyll` は、GitHub Pages において、アンダースコア `_` から始まるファイルを無視しないようにするために必要なファイル
